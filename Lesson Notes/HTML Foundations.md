# HTML Foundations

## Introduction to HTML and CSS
**What do HTML and CSS stand for?** HTML stands for HyperText Markup Language. CSS stands for Cascading Style Sheets

**Between HTML and CSS, which would you use for putting paragraphs of text on a webpage?** You would use HTML as CSS is for the styling of the webpage.

**Between HTML and CSS, which would you use for changing the font and background color of a button?** You would use CSS as that controls the styling of the page.

**What is the difference between HTML, CSS and JavaScript?**
HTML: Creates the structure of a website.
CSS: Stylises the website.
JavaScript: Increases the interactivity of the website.

## HTML Boilerplate
``` HTML
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Website</title>
  <meta name="description" content="A comprehensive tutorial template for learning HTML5 and CSS basics.">
  <meta name="author" content="Your Name">
  <link rel="stylesheet" href="style.css">
  <link rel="icon" type="image/x-icon" href="favicon.ico">
  </head>

<body>
  <header>
    <h1>Welcome to My Website!</h1>
    <p>This is a structured, styled HTML5 template.</p>
  </header>

  <main>
    <section>
      <h2>Why these tags matter</h2>
      <p>The code hidden inside the <code>&lt;head&gt;</code> tag tells the browser how to render the page correctly, optimize it for mobile screens, and read external styling rules.</p>
    </section>
  </main>

  <footer>
    <p>&copy; gekkzzz (me)</p>
  </footer>

</body>
</html>
```
